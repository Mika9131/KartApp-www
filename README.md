## Features
- [x] login
- [x] registration
- [x] resend password
- [x] manage cardsets/cards
- [x] learn-function
- [x] show statistics
- [x] manage friendlist
- [x] share cardsets
- [ ] intigrate friends to share-function
- [ ] accept function in freindlist
- [ ] limit of cardsets/cards
